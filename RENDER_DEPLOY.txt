RENDER DEPLOYMENT - FastAPI Backend (FREE)

1. CREATE ACCOUNT
   https://dashboard.render.com
   - Sign up with GitHub (free, no credit card)

2. DEPLOY WEB SERVICE
   - Click "New +" → "Web Service"
   - Connect GitHub → Select: ymzpenet-beep/ASR_SALES_CALLS_AGENT
   - Settings:
     * Name: ivrimeet-backend (or any name)
     * Environment: Python 3
     * Build Command: pip install -r requirements.txt
     * Start Command: uvicorn agent_service.api:app --host 0.0.0.0 --port $PORT
     * Plan: Free

3. ADD ENVIRONMENT VARIABLES
   Click "Environment" tab and add:
   
   DATABASE_URL=postgresql://user:password@ep-xxx.aws.neon.tech/ivrimeet_db?sslmode=require
   RUNPOD_ENDPOINT_ID=ep-xxxxx
   RUNPOD_API_KEY=your_runpod_api_key
   RUNPOD_SERVERLESS_URL=https://your-endpoint.runpod.io
   IVRIT_API_KEY=your_ivrit_api_key
   LLM_API_KEY=your_deepseek_api_key
   JWT_SECRET_KEY=your_secret_min_32_chars
   USE_RUNPOD=true
   CORS_ORIGINS=https://your-app.netlify.app
   ENVIRONMENT=production

4. DEPLOY
   - Click "Create Web Service"
   - Wait 5-10 minutes for build
   - Get your URL: https://ivrimeet-backend.onrender.com

5. USE IN NETLIFY
   Add to Netlify environment variables:
   VITE_API_BASE_URL=https://ivrimeet-backend.onrender.com
